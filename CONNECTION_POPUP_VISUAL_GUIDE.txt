POPUP STATES & COLORS
═══════════════════════════════════════════════════════════════════════════════════

STATE 1: CONNECTING (YELLOW)
───────────────────────────

┌──────────────────────────────────────────┐
│  🔄 Connecting to Backend...             │ ← Yellow/Orange Color
│                                          │
│  Appears when sending request            │
│  Does NOT auto-hide (stays until done)   │
└──────────────────────────────────────────┘

Example triggers:
  • User clicks "Create Student" button
  • GenerateTeachingContent() called
  • GenerateQuiz() called
  • Any network request initiated


STATE 2: CONNECTED (GREEN)
──────────────────────────

┌──────────────────────────────────────────┐
│  ✅ Connected to http://localhost:8000   │ ← Green Color
│                                          │
│  Appears on successful connection        │
│  Auto-hides in 5 seconds                 │
└──────────────────────────────────────────┘

Example triggers:
  • Server health check passes
  • Connection established


STATE 3: ERROR (RED)
───────────────────

┌──────────────────────────────────────────┐
│  ❌ Disconnected from Backend            │ ← Red Color
│                                          │
│  Appears when connection fails           │
│  Auto-hides in 5 seconds                 │
└──────────────────────────────────────────┘

Example triggers:
  • Cannot reach server (not running)
  • Network error
  • Request timeout
  • HTTP error (404, 500, etc.)


STATE 4: SUCCESS (GREEN)
───────────────────────

┌──────────────────────────────────────────┐
│  ✓ Student 'John' Created                │ ← Green Color
│  ✓ Lesson Generated: PROJECTILE_MOTION   │
│  ✓ Quiz Generated: Atomic Structure      │
│                                          │
│  Auto-hides in 5 seconds                 │
└──────────────────────────────────────────┘

Example triggers:
  • CreateStudent() succeeds
  • GenerateTeachingContent() succeeds
  • GenerateQuiz() succeeds
  • Any successful operation


STATE 5: WARNING (ORANGE)
────────────────────────

┌──────────────────────────────────────────┐
│  ⚠️ No student ID. Create student first. │ ← Orange Color
│  ⚠️ Session Reset                        │
│                                          │
│  Auto-hides in 5 seconds                 │
└──────────────────────────────────────────┘

Example triggers:
  • Trying to generate lesson without student
  • Session reset
  • Non-critical issues

TROUBLESHOOTING GUIDE
═══════════════════════════════════════════════════════════════════════════════════

PROBLEM: Popup never appears
────────────────────────────
Check:
  1. Is Canvas visible? (hierarchy panel)
  2. Is ConnectionStatusPanel active?
  3. Is StatusText child visible?
  4. Is ConnectionStatusUI script attached?
  5. Are references assigned in Inspector?

Fix: Follow CONNECTION_POPUP_SETUP.txt again


PROBLEM: Popup appears but text is invisible
─────────────────────────────────────────────
Check:
  1. Is TextMeshPro text color WHITE?
  2. Is font size large enough? (36+ recommended)
  3. Is text alignment CENTER?
  4. Is background transparent enough to see text?

Fix: Adjust text color to white (255, 255, 255)


PROBLEM: Popup stays forever (doesn't auto-hide)
────────────────────────────────────────────────
Check:
  1. Auto Hide Time setting in Inspector
  2. Is it set to 0? (0 = no auto-hide)

Fix: Set Auto Hide Time to 5 (or your preference)


PROBLEM: Server connected but operations fail
──────────────────────────────────────────────
Check:
  1. FastAPI server running? (python main.py)
  2. Correct server URL? (http://localhost:8000)
  3. Student created before using lessons?
  4. Check backend console for errors

Fix: Restart server, create student first


PROBLEM: Connection check shows red (disconnected)
──────────────────────────────────────────────────
Check:
  1. FastAPI server running?
     Terminal: python main.py
  
  2. Port 8000 accessible?
     Browser: http://localhost:8000/docs
  
  3. Firewall blocking?
     Windows Firewall: Allow Python port 8000
  
  4. Network connectivity?
     Check ping: ping localhost

Fix: Start FastAPI server, check firewall

QUICK START AGAIN (TL;DR)
═══════════════════════════════════════════════════════════════════════════════════

1. Copy ConnectionStatusUI.cs → Assets/Scripts/

2. In Unity:
   - Create Canvas + Panel for status display
   - Add ConnectionStatusUI script to panel
   - Assign StatusText in Inspector
   - Attach EnhancedVRBackendConnector to BackendManager
   - Connect the status UI

3. Test:
   - Press Play
   - Look at top-center for popup
   - Check console for connection status

4. Done! Popups show automatically ✅
