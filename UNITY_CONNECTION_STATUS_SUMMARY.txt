⚡ 2-MINUTE QUICK START
═══════════════════════════════════════════════════════════════════════════════════

1. COPY SCRIPT

   └─ Press 'D' for diagnostic panel

🚀 BASIC WORKFLOW
═══════════════════════════════════════════════════════════════════════════════════

In Your Game Script:

    public class GameController : MonoBehaviour
    {
        void Start()
        {
            var connector = FindObjectOfType<EnhancedVRBackendConnector>();
            
            // Popups will show automatically!
            connector.CreateStudent("John", "john@school.com", 10);
            
            // Subscribe for extra handling if needed
            connector.OnStudentCreated += (response) => 
            {
                Debug.Log("Student created: " + response);
            };
            
            connector.OnError += (error) =>
            {
                Debug.LogError("Error: " + error);
            };
        }
    }

No extra code needed for popups - they show automatically! ✅

🆘 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════════

Problem                          Solution
─────────────────────────────────────────────────────────────────────────────────

Popup not appearing             • Check Canvas is visible
                                • Verify ConnectionStatusUI attached
                                • Check references in Inspector

Text not visible                • Set text color to WHITE
                                • Increase font size
                                • Make background less transparent

Always shows "disconnected"     • Start backend: python main.py
                                • Check port 8000 is open
                                • Verify URL in script

Operations fail silently        • Check console for errors
                                • Enable diagnostic mode (press D)
                                • Check backend console

Popup not auto-hiding           • Set Auto Hide Time > 0
                                • Check CanvasGroup alpha

Can't find connector            • Make sure EnhancedVRBackendConnector
                                  is in scene
                                • Use FindObjectOfType<>()

🎯 COMMON NEXT QUESTIONS
═══════════════════════════════════════════════════════════════════════════════════

Q: Can I use this on a remote server?
A: Yes! Change URL from localhost to server IP
   Example: http://192.168.1.100:8000

Q: Can I customize the popup appearance?
A: Yes! Change colors, position, size in Canvas setup

Q: Do I need to manually handle all popups?
A: No! EnhancedVRBackendConnector handles them automatically

Q: Can I disable popups?
A: Yes! Use the original VRBackendConnector instead
   Or set Show On Connect/Error to false

Q: How do I debug connection issues?
A: Press 'D' for diagnostic panel, check console logs

Q: Can I use this with other UI systems?
A: Yes! Adjust Canvas settings for your UI framework
