📋 SYSTEM OVERVIEW
═══════════════════════════════════════════════════════════════════════════════════

Three-Layer Architecture:

LAYER 1: UI Display (ConnectionStatusUI)
    ├─ Displays colored popup messages
    ├─ Handles auto-hide timers
    ├─ Color-coded: Green (success), Red (error), Yellow (waiting)
    └─ Location: Top-center of screen

LAYER 2: Connection Handler (EnhancedVRBackendConnector)
    ├─ Manages all HTTP requests
    ├─ Triggers UI updates automatically
    ├─ Handles timeouts and errors
    └─ Provides events for game logic

LAYER 3: Game Integration (Your Game Script)
    ├─ Calls connector methods
    ├─ Subscribes to events (optional)
    ├─ Updates game based on responses
    └─ Automatically sees popups
